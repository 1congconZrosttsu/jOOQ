### Description

This example uses

1. [R2dbc](https://r2dbc.io/)

### Installation

To install and run this example, simply check it out and follow these steps

1. Run the following commands

```
$ pwd
/path/to/checkout/dir
$ cd jOOQ-examples/jOOQ-r2dbc-example
...
$ mvn clean install
...
```
